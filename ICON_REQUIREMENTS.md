# ShortsBlock Icon Requirements

- **16x16 px** - Browser toolbar (small icon)
- **48x48 px** - Extension management page (required)
- **96x96 px** - Extension management page (required)
- **128x128 px** - Chrome Web Store listing (recommended)
- **256x256 px** - Chrome Web Store listing (recommended)

- **48x48 px** - Toolbar icon (required)
- **96x96 px** - Extension management page (required)
- **128x128 px** - AMO listing (recommended)

### Safari Extension Icons
- **16x16 px** - Toolbar icon
- **32x32 px** - Toolbar icon (retina)
- **48x48 px** - Extension management
- **64x64 px** - Extension management (retina)
- **128x128 px** - App Store listing
- **256x256 px** - App Store listing (retina)
- **512x512 px** - App Store listing (retina)

### Edge Add-ons Store Icons
- **300x300 px** - Store listing (required)

- **1400x560 px** - Marquee promo tile

### Chrome Web Store Icons

- **440x280 px** - Small promo tile

- **1400x560 px** - Marquee promo tile (alternative)

## Icon Design Guidelines

### Branding Requirements
- **NO YouTube Shorts logo** - Must be original design
- **NO copyrighted material** - Original artwork only
- **Color scheme**: Choose brand colors (suggest: red/block theme, or blue/focus theme)
- **Style**: Modern, clean, minimalist
- **Concept**: Should represent "blocking" or "stopping" Shorts

### Design Suggestions
- Block/stop sign icon
- Shield with slash
- "No" symbol (circle with diagonal line)
- Custom "SB" or "ShortsBlock" monogram
- Minimalist geometric design

### Technical Requirements
- **Format**: PNG with transparency (alpha channel)
- **Background**: Transparent or solid color
- **File size**: Optimized (under 50KB for small icons, under 200KB for large)
- **Quality**: High resolution, crisp edges
- **Consistency**: Same design across all sizes

## File Naming Convention

All icons should follow this naming:
- `icon-{size}.png` for standard icons (e.g., `icon-48.png`)
- `store-{description}-{size}.png` for store-specific icons
- `icon.svg` for vector format

## Icon Locations
